PLANTAGOCHI: Vaso Inteligente Interativo

O Plantagochi visa auxiliar na rega adequada, transformando o cuidado com a planta em uma interação divertida. Ele monitora a umidade do solo e fornece feedback visual para garantir a saúde da planta. O projeto se insere na área de IoT básica e Automação Residencial.

Passo a Passo da Montagem

Instruções claras e detalhadas para replicação do projeto, incluindo imagens em cada passo.

  1. Conexão dos LEDs

    Ligue os LEDs (com resistores) aos pinos digitais do Arduino (Ex: D2, D3, D4).

  2. Conexão do Sensor de Umidade

    Conecte o pino analógico do sensor à porta A0 do Arduino.

Código do Projeto

Trechos de código essenciais (snippets) com formatação adequada e explicação das partes importantes.

Snippet: Leitura do Sensor e Lógica dos LEDs

                
                    // Variáveis de definição de pinos
                    #define PINO_SENSOR A0
                    #define LED_VERMELHO 2
                    // ...
                    
                    void loop() {
                        int umidade = analogRead(PINO_SENSOR);
                        
                        if (umidade < 300) {
                            // Planta com sede: LED Vermelho
                            digitalWrite(LED_VERMELHO, HIGH);
                        } 
                        // ...
                    }

Explicação: A função analogRead() lê o valor do sensor de umidade, que é então comparado com limiares definidos para acionar o LED correspondente, indicando o estado da planta.

Problemas e Lições Aprendidas

Desafios Enfrentados

  • Problema: A bomba não acionava corretamente com o 5V do Arduino.
  • Resolução: Foi necessário incorporar um módulo relé e uma fonte de alimentação externa para a bomba.

Principais Lições

Aprendeu-se a importância da correta **calibração de sensores** e o uso estratégico de **componentes de acionamento** (relés) para lidar com atuadores de maior potência.
